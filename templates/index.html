About the Project

Green Alert is an innovative platform designed to provide timely, verified environmental alerts and actionable recommendations to vulnerable communities. By leveraging AI for analysis and blockchain for data integrity, Green Alert empowers users with localized climate insights, enabling climate-resilient decision-making.

The system delivers alerts via SMS, Email, and a Web Dashboard, ensuring accessibility for all users. It integrates real-time weather data, AI-driven recommendations, and blockchain-backed transparency to create a reliable and impactful climate adaptation tool.
